📖 မိတ်ဆက် (Introduction)
ဤ Project သည် အသုံးပြုသူများအတွက် ရွှေဈေးနှုန်းနှင့် ငွေပမာဏကို အပြန်အလှန် အလွယ်တကူ တွက်ချက်နိုင်ရန် ရည်ရွယ်ပါသည်။ Single HTML file အမျိုးအစားဖြစ်သောကြောင့် မည်သည့် Browser တွင်မဆို အင်တာနက်မလိုဘဲ အသုံးပြုနိုင်ပါသည်။ [cite: 2026-02-07]
✨ အဓိက လုပ်ဆောင်ချက်များ (Key Features)
 * ကျပ်ငွေမှ ရွှေအလေးချိန်သို ပြောင်းလဲခြင်း: မိမိဝယ်ယူလိုသော ငွေပမာဏရိုက်ထည့်ရုံဖြင့် ရရှိမည့် ကျပ်၊ ပဲ၊ ရွေး ကို အတိအကျ သိရှိနိုင်ခြင်း။ [cite: 2026-02-07]
 * ရွှေအလေးချိန်မှ ကျပ်ငွေသို ပြောင်းလဲခြင်း: မိမိမှာရှိသော ရွှေအလေးချိန်အလိုက် ကျသင့်မည့် ငွေပမာဏကို တွက်ချက်နိုင်ခြင်း။ [cite: 2026-02-07]
 * Real-time Calculation: ဂဏန်းများ ရိုက်ထည့်သည်နှင့် တစ်ပြိုင်နက် အဖြေထွက်ပေါ်ခြင်း။ [cite: 2026-02-07]
 * Responsive Design: ဖုန်း၊ တက်ဘလက်နှင့် ကွန်ပျူတာအားလုံးတွင် ကြည့်ကောင်းအောင် Dark Mode ဖြင့် ဖန်တီးထားခြင်း။ [cite: 2026-02-07]
🛠 အသုံးပြုထားသော နည်းပညာများ (Technologies Used)
 * HTML5 - တည်ဆောက်ပုံအတွက်။
 * CSS3 - လှပသော UI နှင့် Dark Theme အတွက်။
 * JavaScript (ES6) - တွက်ချက်မှုဆိုင်ရာ Logic များအတွက်။
📐 တွက်ချက်ပုံစံ (Calculation Logic)
